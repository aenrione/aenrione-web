# aenrione.xyz

Personal blog by **Alfredo Enrione**. Built with **Astro** using the **Fuwari** theme and customized for a bilingual (ES/EN) workflow.

## ✨ Features
- Bilingual posts (ES/EN) with language switcher
- Search (Pagefind)
- Light/Dark mode
- Tags & categories
- Responsive layout

## 🚀 Local development
```bash
pnpm install
pnpm dev
```
Open: http://localhost:4321

## 🏗️ Build & run
```bash
pnpm build
pnpm start
```

## 🐳 Docker (Portainer)
```bash
docker compose up -d --build
```
App runs on **port 4321**.
